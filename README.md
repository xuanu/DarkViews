# DarkViews
default pressed view

> 带默认按下效果的控件
> [添加依赖方法](https://jitpack.io/#xuanu/DarkViews/v1.0)
> 拥有控件
```
DarkButton extends Button
DarkImage extends ImageView
DarkRadio extends RadioButton
DarkText extends TextView
```
> 使用方法同继承控件，仅仅默认加了按下效果 。
